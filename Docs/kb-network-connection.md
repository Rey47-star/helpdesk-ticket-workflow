# Knowledge Base Article: Troubleshooting Network Connection Issues

This KB article provides steps to troubleshoot common network connectivity problems.

---

## Symptom: Unable to Connect to Network

### Possible Causes:
- Incorrect network credentials  
- Network adapter issues  
- Router or modem problems  
- IP address conflicts

---

## Step-by-Step Troubleshooting:

1. **Verify Network Credentials**  
   Ensure the correct Wi-Fi password is used.  
   Try connecting another device to confirm the network is working.

2. **Restart Network Devices**  
   Power cycle the computer, router, and modem.  
   Wait 30 seconds before turning them back on.

3. **Check Network Adapter**  
   Open Device Manager (Windows) or System Preferences (Mac) to ensure the network adapter is enabled and functioning.

4. **Release and Renew IP Address**  
   On Windows: `ipconfig /release` then `ipconfig /renew` in Command Prompt.  
   On Mac: Use Network Preferences to renew DHCP lease.

5. **Check for Interference or Signal Issues**  
   Move closer to the router or reduce physical obstructions.  
   Consider switching Wi-Fi channels.

---

## Escalation
If the issue persists, escalate to network support or consider hardware replacement.
